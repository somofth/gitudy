

### 🎮 Git-Vis 퀴즈 시나리오 (Total 5 Levels)

#### **Level 1. 창고 개방 (git init)**

> "모든 일의 시작은 창고 정리부터!"

* **상황:** 아직 아무런 시스템이 없는 상태. 화면의 네 가지 공간(방, 현관, 창고, 구름)이 전부 회색으로 비활성화되어 있음.
* **지문:** "새로운 프로젝트를 시작하려고 해! 이 폴더를 Git이 관리하도록 설정해줄래?"
* **시각적 단서:** 방(Working Directory)에 불이 꺼져 있음.
* **선택지:** `git init` | `git start` | `git open` | `git help`
* **정답:** **`git init`**
* **성공 피드백:**
* (애니메이션) 💡 **탁!** 하고 방, 현관, 창고에 불이 들어오며 활성화됨.
* (메시지) "좋아! 이제 이 공간은 Git 감시 하에 있어. 관리자가 배치됐어!"



---

#### **Level 2. 이삿짐 담기 (git add)**

> "바닥에 널브러진 서류, 일단 박스에 담자."

* **상황:** [내 방] 바닥에 빨간색 서류(파일)가 하나 생김. `index.html`이라고 적혀 있음.
* **지문:** "코드를 다 짰어! (수정 완료). 이걸 기록으로 남기려면, 우선 **현관에 있는 박스(Staging Area)**에 넣어야 해."
* **시각적 단서:** [내 방]에 빨간색 파일이 반짝거림. [현관]에는 빈 박스가 있음.
* **선택지:** `git commit` | `git push` | `git add` | `git move`
* **정답:** **`git add`**
* **오답 유도(함정):** `git commit`을 누르면 -> "📦 박스에 담지도 않았는데 포장할 순 없어! 먼저 담아줘."
* **성공 피드백:**
* (애니메이션) 서류가 슝 날아가서 현관 박스에 쏙 들어감. 파일 색이 초록색으로 변함.
* (메시지) "나이스 샷! 이제 포장할 물건을 다 골랐어."



---

#### **Level 3. 포장 및 기록 (git commit)**

> "박스를 테이프로 밀봉하고 견출지 붙이기."

* **상황:** [현관]에 서류가 담긴 초록색 박스가 놓여 있음. 아직 뚜껑은 열려 있음.
* **지문:** "박스에 물건을 다 담았으니, 이제 뚜껑을 닫고 **'첫 번째 버전'이라고 라벨을 붙여서** 창고에 넣자."
* **시각적 단서:** [현관]의 박스가 대기 중. [창고]는 비어 있음.
* **선택지:** `git save` | `git commit` | `git status` | `git add`
* **정답:** **`git commit`**
* **성공 피드백:**
* (애니메이션) 박스가 테이프로 칭칭 감기고(포장), [집 앞 창고]로 이동해서 '착' 하고 쌓임. 파란색으로 변함.
* (메시지) "기록 완료! 언제든 이 상태로 시간을 되돌릴 수 있게 됐어."



---

#### **Level 4. 물류센터로 발송 (git push)**

> "내 컴퓨터가 고장 나면 어떡해? 구름 위로 보내자!"

* **상황:** [집 앞 창고]에 파란색 박스가 하나 쌓여 있음. [구름 위 물류센터]는 텅 비어 있음.
* **지문:** "내 컴퓨터에만 두면 불안해. 친구들도 볼 수 있게 **원격 저장소(GitHub)**로 트럭을 보내볼까?"
* **시각적 단서:** 창고 앞에 트럭이 시동을 걸고 대기 중.
* **선택지:** `git upload` | `git pull` | `git push` | `git send`
* **정답:** **`git push`**
* **오답 유도(함정):** `git pull`을 누르면 -> "🚚 엥? 물건을 보내야지 가져오면 안 되지!"
* **성공 피드백:**
* (애니메이션) 트럭이 박스를 싣고 구름 위로 슈웅 날아감. 구름 위에 보라색 박스가 생김.
* (메시지) "발송 성공! 이제 전 세계 어디서든 네 코드를 볼 수 있어."



---

#### **Level 5. 친구의 작업물 가져오기 (git pull)**

> "친구가 작업을 끝냈대. 내 방으로 가져오자."

* **상황:** (갑자기 알림음 띠링!) [구름 위 물류센터]에 내가 모르는 '노란색 박스'가 새로 생김. [내 방]에는 그 박스가 없음.
* **지문:** "어? 팀원이 새 기능을 만들어서 올렸대! 최신 내용을 **내 방으로 가져와서** 합쳐야 해."
* **시각적 단서:** 구름 위에서 내 방으로 연결된 파이프가 반짝거림.
* **선택지:** `git clone` | `git push` | `git fetch` | `git pull`
* **정답:** **`git pull`**
* **참고:** `git clone`도 헷갈릴 수 있는데, 여기선 "이미 작업 중인 프로젝트의 변경 사항 가져오기"이므로 `pull`이 맞음. 오답 힌트로 "clone은 아예 처음 복제할 때 쓰는 거야!"라고 알려주면 됨.
* **성공 피드백:**
* (애니메이션) 구름에서 박스가 낙하산 타고 내려와 [내 방], [창고]에 동시에 적용됨.
* (메시지) "동기화 완료! 이제 친구가 만든 코드 위에서 계속 작업할 수 있어."

### 🎁 클리어 보상 (Gamification, 선택)

모든 퀴즈를 맞히면, 마지막에 사용자의 성향을 분석한 **'개발자 유형 카드'**를 보여주면 공유하고 싶어질 거야.

* **"당신은... 꼼꼼한 이삿짐 센터 소장님!"**
* (한 번도 안 틀렸을 때) "Git 천재의 재능이 보이네요."
* **[다시 하기]** / **[친구에게 자랑하기(공유)]** 버튼 제공.
